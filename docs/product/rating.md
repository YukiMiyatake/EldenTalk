# レーティングシステム

## 概要 
ゲームシステムのレーティングを使わない内戦（カスタムマッチ）向け  
まずはLoL、AoEあたりをターゲットにする  
最初はWebサービスだが、Discordと連携したい  

## 機能  

### Ver0.1  
まずは管理人は一人で管理人が全て登録する  
  
```uiflow

```
